# ONLINE STORE

> An Online Store website developed using Html, CSS, Tailwind, MySQL, Laravel, Livewire

## Features

- **Admin Panel**
- **User Profile**
- **Shoping Cart with Coupon**
- **API**

## Used Packages

- **Livewire**
- **Voyager**
- **Breeze**
- **jwt-auth**
- **Laravel PDF**
- **daisyUI**
- **SweetAlert2**
